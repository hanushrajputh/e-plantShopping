# 🌿 Paradise Nursery (e-plantShopping)

A React + Redux Toolkit shopping cart application for buying houseplants.

## Features
- Browse plants by 3 categories (Indoor, Outdoor, Succulents)
- 6 plants per category (18 total) with thumbnails, names, prices
- Add to Cart (button disables after adding)
- Navbar on all pages (Home, Plants, Cart) with dynamic cart count
- Cart page with quantity +/- controls, delete item, per-item total, and total cart amount
- Continue Shopping and Checkout (Coming Soon)

## Tech Stack
- React (Vite)
- Redux Toolkit
- CSS
